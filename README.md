# 🤖 Smart AI LMS – AI Powered Learning Management System

Smart AI LMS is a modern educational platform that combines course management, AI-assisted learning, and quiz generation into a single responsive application. The platform enables students to access learning resources, interact with an AI assistant, and evaluate their knowledge through automated quizzes.

The system is designed using a modern Full-Stack architecture with React.js, Node.js, Express.js, and MongoDB, providing a scalable and user-friendly learning experience across devices.

---

# 🚀 Key Architectural Capabilities

### 1. 📚 Course Management System
- Add and manage course videos using YouTube links
- Centralized learning resource management
- Easy access to educational content

### 2. 🤖 AI Learning Assistant
- Provides learning guidance and support
- Answers student queries
- Recommends learning resources

### 3. 📝 AI Quiz Generation Engine
- Generates quizzes dynamically
- Supports multiple subjects
- Difficulty-based assessments
- Built-in countdown timer

### 4. 🔐 Authentication & User Management
- User Registration
- Secure Login System
- Role-Based Access Control

### 5. 📱 Responsive User Interface
- Mobile-Friendly Design
- Modern Dashboard
- Optimized User Experience
