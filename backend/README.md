# Smart AI LMS

AI-powered Learning Management System built using MERN Stack + OpenRouter AI.

## Features

- AI Tutor
- Course Management
- Student Dashboard
- Progress Tracking
- Role-based Authentication
- Smart Coding Help

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- OpenRouter AI

## How to Run Project

### 1. Clone Project

git clone YOUR_GITHUB_LINK

### 2. Install Dependencies

Frontend:

cd frontend
npm install

Backend:

cd backend
npm install

### 3. Create .env file inside backend

Add:

OPENROUTER_API_KEY=your_key

MONGO_URI=your_mongodb_url

JWT_SECRET=your_secret

PORT=5000

### 4. Start Backend

npm start

### 5. Start Frontend

npm run dev
